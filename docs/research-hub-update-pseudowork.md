# Research Hub 改修指示書：「ブルシットジョブの手前にあるもの」公開に伴う全体整合

## 概要

`essay-pseudowork.html` が公開済みである。これに伴い、Research Hub の以下4ファイルを改修する。

**対象ファイル（すべてリポジトリルート直下）：**
1. `index.html` — Home ページ
2. `overview.html` — Overview ページ
3. `applications.html` — Applications ページ
4. `essay-challenge.html` — 既存エッセイ（max-width 修正のみ）

**改修は全7項目。各項目ごとに、何を、どこに、どう変更するかを完全に記述する。**

---

## 改修手順

**重要：各ファイルを編集する前に必ず現在の内容を読み、既存の HTML 構造（クラス名、タグ構成、インデント）を確認すること。新規追加は既存要素と完全に同じ HTML 構造・クラスで行うこと。推測で書かず、実物を見て合わせること。**

## やってはいけないこと

- 既存の HTML 構造やクラス名を変更・リファクタリングしないこと
- 既存のテキスト内容を勝手に変更しないこと（指示された箇所のみ）
- 新しい CSS クラスを作らないこと（既存クラスを再利用する）
- CSS を外部ファイルに分離しないこと
- ナビゲーション（ヘッダー、フッター）の構造を変更しないこと
- essay-pseudowork.html の内容を変更しないこと（既に公開済み）
- essay-bivalence.html の内容を変更しないこと
